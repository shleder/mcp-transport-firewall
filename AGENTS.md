Agent Context for MCP-server

Этот проект является Fail-Closed прокси-шлюзом для Model Context Protocol.

Архитектурный стиль: Функциональное программирование, TypeScript ES2022, Strict Mode.

Безопасность: Жесткая реализация MCP Colors (запрет смешивания Red и Blue инструментов).

Обработка ошибок: Только Hard Halt. Запрещен возврат null/undefined при сбоях валидации.
